# 🚗 Car Rental System (C++)

The **Car Rental System** is a console-based application developed using **C++**.
It is designed to manage the basic operations of a car rental service such as displaying available cars, booking a car, and managing rental details.

This project demonstrates the use of **object-oriented programming concepts and file handling in C++**.

---

## 📚 Features

* View available cars
* Rent a car
* Display car details
* Manage rental records
* Simple and interactive console interface

---

## 🛠️ Technologies Used

* **C++**
* Object-Oriented Programming (OOP)
* File Handling
* Console-based Interface

---

## ⚙️ How to Run the Project

1. Compile the program using a C++ compiler:

g++ main.cpp -o car_rental

2. Run the program:

./car_rental
